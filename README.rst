Installation
============

You can install emqx-extension-python-sdk either via the Python Package Index (PyPI) or from source.

To install using `setup`:

.. sourcecode:: console

    $ python setup.py develop

To install using `pip`:

.. sourcecode:: console

    $ pip install -U emqx_sdk
